# Digital Literacy Project – Student Digital Ambassador

##  Project Overview
This project was completed as part of the CSE0001 – Digital Literacy course to develop essential digital skills required in today’s technology-driven world. The project follows the role of a Student Digital Ambassador, focusing on helping students use digital tools responsibly and professionally.

##  Objectives
- Promote digital awareness
- Build a professional online presence
- Learn collaboration and coding platforms
- Practice professional email communication
- Understand cyber safety and cybercrime prevention

##  Project Tasks

### Task 1 – Digital Awareness
Created digital awareness content and posters using Canva to promote responsible internet usage.

### Task 2 – Professional Profile
Designed a professional digital profile highlighting qualifications, skills, and achievements.

### Task 3 – Platform Exploration
Explored GitHub and collaborative platforms to understand version control and online project management.

### Task 4 – Email Etiquette
Practiced writing professional emails including internship inquiries and academic requests.

### Task 5 – Cyber Safety Awareness
Developed cyber safety resources explaining online threats and prevention methods.

##  Tools Used
- Canva
- GitHub
- Gmail
- Online Collaboration Platforms

##  Learning Outcomes
- Improved digital communication skills
- Better understanding of professional online behavior
- Awareness of cyber safety practices
- Experience with GitHub repositories
